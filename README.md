## 运行

注意：本项目仅适用于学习和研究，不得用于商业使用,或者做违法事情。

### 本地开发

```bash
git clone git@github.com:susakin/threads-react.git

cd threads-react
yarn
yarn start
```

打开浏览器并访问: [http://127.0.0.1:3001](http://127.0.0.1:3001)
